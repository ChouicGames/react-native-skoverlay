# react-native-skoverlay

Expose SKOverlay for react-native

## Installation

```sh
npm install react-native-skoverlay
```

## Usage

```js
import { showOverlay } from 'react-native-skoverlay';

// ...

const result = await showOverlay(3, 7);
```
